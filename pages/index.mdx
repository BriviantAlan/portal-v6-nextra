# Project Overview

## Introduction
Welcome to the documentation for **Portal V6**. This project is designed to provide users with comprehensive access and management capabilities for servers under their purchased subscriptions. Portal V6 allows users to manage their subscriptions, control server operations, and connect to their servers with ease.

## Key Features
| Feature                  | Description |
|--------------------------|-------------|
| **Server Access**        | Users can access the servers associated with their purchased subscriptions. |
| **Subscription Management** | Users can update or cancel their subscriptions as needed. |
| **Server Control**       | Users can restart their servers directly from the app. |
| **Connection Files**     | Users can download connection files for easy access to their servers. |
| **Application Installation Requests** | Users can request the installation of applications on their servers. |
| **Server Connectivity**  | Users can establish connections to their servers efficiently. |
## Tech Stack
This project is built using the following technologies:
- **Next.js**: A React framework for building server-side rendered and static web applications.
- **React**: A JavaScript library for building user interfaces.
- **Node.js**: A JavaScript runtime built on Chrome's V8 JavaScript engine.

## Important Libraries
- **[Shadcn UI](https://ui.shadcn.com/)**: A component library used for building user interfaces.
- **[Auth0](https://auth0.com/)**: A library for authentication and authorization.
- **[Recurly](https://recurly.com/)**: A subscription billing management library.
